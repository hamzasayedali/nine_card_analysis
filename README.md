# nine_card_analysis
Analysis of the best strategies for the nine card game

# Project Goal
- make a visual simulation of the Nine Card card game in python
- make an API to play the game
- make a few strategy functions that take in the current state of the game and choose a next move
- perform a statistical analysis of the different strategies and find which have better odds of winning
- find out some good options for human-playable strategies that don't involve too much thinking
